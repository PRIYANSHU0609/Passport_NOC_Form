# 🛂 Passport NOC Application Form — NALCO

A responsive, interactive web form developed during a summer internship at **National Aluminium Company Limited (NALCO), Bhubaneswar**. The form allows employees to apply for a **No Objection Certificate (NOC) from the Vigilance Cell** for obtaining a passport.

## 🚀 Features

- **Structured Multi-Section Form** — Collects personal details, designation, department, property declarations, and travel information in a clean, organized layout.
- **Toggle-Switch Conditional Inputs** — Dynamic Yes/No toggle switches that show or hide additional input fields based on user selection:
  - *"Whether application made earlier"* → reveals date and recipient field
  - *"Whether visited foreign country earlier"* → reveals additional travel details
- **Certification Checkbox** — Employee must certify the information before submitting.
- **Clear & Submit Actions** — Reset form or submit with dedicated styled buttons.
- **Responsive Design** — Fully functional across desktop and tablet screen sizes using CSS Flexbox.
